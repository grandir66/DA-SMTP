Connessione & API

Configurazione del canale tra il listener relay (daemon SMTP, processo separato) e questo admin standalone. Il listener legge regole/template/clienti da qui via HTTPS e pushia eventi in tempo reale.

API key

Chiave da inserire come STORMSHIELD_RELAY_API_KEY (o DOMARC_RELAY_API_KEY) nel secrets.env del listener:

{{ api_key }}

Configurazione listener (esempio)

# /etc/stormshield-smtp-relay/secrets.env (o equivalente)
RELAY_API_KEY={{ api_key }}
RELAY_MANAGER_BASE_URL=https://192.168.4.41:8443

# /etc/stormshield-smtp-relay/relay.yaml
manager:
  base_url: "https://192.168.4.41:8443"
  api_key_env: "RELAY_API_KEY"
  ca_bundle_env: "RELAY_MANAGER_CA_BUNDLE"  # path al cert.crt o vuoto+verify_tls=false in dev
  verify_tls: false   # cert self-signed in dev
