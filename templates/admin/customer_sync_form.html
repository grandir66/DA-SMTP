Esempio query SQL per JOIN domini/aliases (quando i domini stanno in una tabella separata, come client_domains): 
SELECT
  c.aconto                                    AS codcli,
  c.ragione_sociale                           AS ragione_sociale,
  STRING_AGG(d.dominio, ',')                  AS dominii_csv,
  STRING_AGG(a.alias_name, ',')               AS aliases_csv
FROM clienti c
LEFT JOIN client_domains d
       ON d.codice_cliente = c.aconto
      AND COALESCE(d.excluded, FALSE) = FALSE
LEFT JOIN customer_aliases a
       ON a.codcli = c.aconto
WHERE COALESCE(c.aescluso, FALSE) = FALSE
GROUP BY c.aconto, c.ragione_sociale

Mapping consigliato: codcli → codcli, ragione_sociale → ragione_sociale, dominii_csv → domains (transform: split:,), aliases_csv → aliases (transform: split:,).
